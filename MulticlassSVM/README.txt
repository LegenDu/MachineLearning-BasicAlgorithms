1. To run code for Multiclass SVM (Wine Dataset):
	a. Open the terminal
	b. Switch directories to this folder
	c. Run the command --> 
        Linear Kernel: python3 MulticlassSVM.py linear
        RBF Kernel: python3 MulticlassSVM.py rbf
    d. Results will be printed to the console

2. To run code for Multiclass SVM (Digits Dataset):
	a. Open the terminal
	b. Switch directories to this folder
	c. Run the command --> 
        Linear Kernel: python3 MulticlassSVM_Digits.py linear
        RBF Kernel: python3 MulticlassSVM_Digits.py rbf
    d. Results will be printed to the console

* It could take a long time to calculate.
