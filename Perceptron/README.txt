1. To run code for Perceptron vs Perceptron(Dual Form) (Perceptron Dataset):
	a. Open the terminal
	b. Switch directories to this folder
	c. Run the command --> python3 Perceptron.py
    d. Results will be printed to the console

2. To run code for Kernelized Perceptron (Two Spirals Dataset):
	a. Open the terminal
	b. Switch directories to this folder
	c. Run the command --> 
        Linear Kernel: python3 Kernelized_Perceptron.py l
        Gaussian (RBF) Kernel: python3 Kernelized_Perceptron.py g
    d. Results will be printed to the console
