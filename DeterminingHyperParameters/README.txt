1. To run code for Determining Model Hyper-parameters by optimizing accuracy(Breast Cancer, Spambase, Diabetes Datasets):
	a. Open the terminal
	b. Switch directories to this folder
	c. Run the command --> 
        Linear Kernel: 
            Breast Cancer: python3 Hyperparameters_Accuracy.py BreastCancer linear
            Spambase: python3 Hyperparameters_Accuracy.py Spambase linear
            Diabetes: python3 Hyperparameters_Accuracy.py Diabetes linear
        Gaussian (RBF) Kernel: 
            Breast Cancer: python3 Hyperparameters_Accuracy.py BreastCancer rbf
            Spambase: python3 Hyperparameters_Accuracy.py Spambase rbf
            Diabetes: python3 Hyperparameters_Accuracy.py Diabetes rbf
    d. Results will be printed to the console
