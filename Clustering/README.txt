1. To run code for K Means Clustering:
	a. Open the terminal
	b. Switch directories to this folder
	c. Run the command --> 
        1) For multiple k
            Dermatology: python3 KMeans.py Dermatology l
            Vowels: python3 KMeans.py Vowels l
            Glass: python3 KMeans.py Glass l
            Ecoli: python3 KMeans.py Ecoli l
            Yeast: python3 KMeans.py Yeast l
            Soybean: python3 KMeans.py Soybean l
        2) For K equals to the number of calsses
            Dermatology: python3 KMeans.py Dermatology c
            Vowels: python3 KMeans.py Vowels c
            Glass: python3 KMeans.py Glass c
            Ecoli: python3 KMeans.py Ecoli c
            Yeast: python3 KMeans.py Yeast c
            Soybean: python3 KMeans.py Soybean c
    d. Results will be printed to the console
