1. To run code for Logistic Regression (Spambase, Breast Cancer, Pima Indian Diabetes Datasets):
	a. Open the terminal
	b. Switch directories to this folder
	c. Run the command -->
        For Spambase dataset: python3 LogisticRegression.py Spambase
        For Breast Cancer dataset: python3 LogisticRegression.py BreastCancer
        For Pima Indian Diabetes dataset: python3 LogisticRegression.py Diabete
    d. Results will be printed to the console, plotted two graphs will show in pop-up windows.
